## tacc-keys

Client for TACC's public ssh keys service

### Synopsis

Client for TACC's public ssh keys service.
    
    It allows you to create, list, publish, and delete public ssh keys from the
    keys service

### Options

```
      --config string   config file (default is $HOME/.tacc-keys.yaml)
  -h, --help            help for tacc-keys
```
